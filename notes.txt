## Product Requirements Document (PRD)

**Product Name:** Personal Life OS
**Type:** Personal Productivity & Learning Management Platform
**User:** Single user (self-hosted)
**Design Style:** Soft Glassmorphism UI

---

# 1. Product Overview

Personal Life OS is a **self-hosted personal platform** used to manage:

* Daily tasks
* Learning roadmap
* Skill tracking
* Habit tracking
* Personal notes
* Productivity analytics

The platform is designed **only for personal use**, allowing full customization and independence from third-party tools like Notion or Trello.

Primary goals:

* Manage daily life and learning in one system
* Track long-term skill development (Data Engineering)
* Generate personal productivity insights from stored data
* Build a lifelong customizable productivity system

---

# 2. Objectives

1. Centralize all personal management activities in one platform.
2. Track learning progress toward becoming a Data Engineer.
3. Maintain a structured roadmap for skills and projects.
4. Enable long-term analytics on personal productivity.
5. Provide a visually calm and premium UI using glassmorphism.

---

# 3. Target User

Single user (the owner of the system).

Usage scenarios:

* Daily task planning
* Managing learning roadmap
* Tracking skill progress
* Recording notes and insights
* Viewing productivity analytics

# 4. API Design

## Task APIs

Create Task

```
POST /tasks
```

Request

```
{
"title": "Learn SQL window functions",
"description": "Practice ranking queries",
"priority": "high",
"due_date": "2026-03-18",
"category": "learning"
}
```

Response

```
{
"id": "uuid",
"status": "created"
}
```

---

Get Tasks

```
GET /tasks
```

Filters:

```
/tasks?status=todo
/tasks?priority=high
/tasks?due=today
```

---

Update Task

```
PUT /tasks/{id}
```

---

Delete Task

```
DELETE /tasks/{id}
```
